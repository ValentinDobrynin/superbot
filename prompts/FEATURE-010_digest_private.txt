ЖЁСТКИЕ ПРАВИЛА (в порядке приоритета):

1. КАЖДАЯ ФРАЗА → ОДНА КАТЕГОРИЯ. Никогда не дублируй один и тот же фрагмент в commitments и events одновременно.

2. COMMITMENT — обещание конкретного действия с глаголом: «заберу», «привезу», «позвоню», «приеду», «закажу», «забронирую», «куплю». Без глагола действия — НЕ commitment.
   ✗ «надо как-нибудь увидеться» — пожелание, не commitment.
   ✗ «классно бы было» — не commitment.
   ✓ «заберу детей с тренировки в 19:00» — commitment.
   ✓ «привезу книгу на выходных» — commitment.

3. EVENT — встреча/событие/поездка с конкретным днём или временем В ИСХОДНОМ ТЕКСТЕ. Не выдумывай время и дату.
   ✗ «давай встретимся как-нибудь» — НЕ event.
   ✓ «ужин у Маши в субботу в 19:00» — event, when_raw="в субботу в 19:00".
   ✓ «летим 12 мая» — event, when_raw="12 мая". НЕ добавляй фейковое «18:00».

4. ВОПРОС (с «?» или со слова «когда/как/почему/удобно/можешь») → open_questions, не commitment.
   - direction "from_me": Я спросил — ответа к концу дня нет.
   - direction "to_me": собеседник спросил — я не ответил.
   - На вопросы, на которые сегодня же ответили, в open_questions не клади.

5. DIRECTION: "from_me" — реплика «Я:»; "to_me" — реплика «{partner_name}:».

6. CLOSED_COMMITMENTS — только то, что сегодня явно закрыто в переписке. Reason="completed" / "cancelled".

7. IS_URGENT: `true` ТОЛЬКО если в исходных сообщениях есть «срочно», «горит», «asap», «немедленно», «сегодня». Без явного маркера — `false`.

8. ПУСТО ЛУЧШЕ ШУМА. Тихий день — пустые массивы `[]`. Не выжимай искусственно «на всякий случай».

Правильный JSON:
```json
{{
  "summary_md": "Договорился забрать Лёву сегодня в 19. Подтвердили субботний ужин у Маши.",
  "commitments": [
    {{ "direction": "from_me", "text": "заберу Лёву с тренировки в 19:00", "deadline_raw": "сегодня 19:00", "is_urgent": true, "source_message_id": null }}
  ],
  "closed_commitments": [],
  "events": [
    {{ "description": "ужин у Маши", "when_raw": "в субботу", "is_urgent": false, "source_message_id": null }}
  ],
  "open_questions": [
    {{ "direction": "from_me", "text": "во сколько начало (ужина у Маши)?" }}
  ]
}}
```

Обрати внимание: «заберу Лёву» попало в commitments один раз. «ужин у Маши» — в events один раз, без выдуманного «19:00». «во сколько начало?» — вопрос, попал в open_questions, не в commitments. is_urgent=true только у первого пункта, потому что в тексте есть «сегодня».
