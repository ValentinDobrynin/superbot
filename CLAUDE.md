# CLAUDE.md

Этот файл — **алиас** на `AGENTS.md`.

Все правила, конвенции и процедуры для AI-агентов (Claude, Cursor, Codex и др.)
описаны в `AGENTS.md`. Чтобы не было двух источников правды, поддерживается
только `AGENTS.md`; этот файл существует исключительно для совместимости с
инструментами, которые ищут именно `CLAUDE.md`.

→ См. [`AGENTS.md`](./AGENTS.md).

Ключевые входные точки для агента:

- [`AGENTS.md`](./AGENTS.md) — правила работы.
- [`project_description.md`](./project_description.md) — что и зачем строим.
- [`backlog.md`](./backlog.md) — текущий список задач.
- [`README.md`](./README.md) — пользовательская документация.
- [`Makefile`](./Makefile) — `make check` как канонический прогон проверок.
